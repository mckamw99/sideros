# context/ — Project Canon & Continuity

> ⚠️ **GM-ONLY MATERIAL.** These files contain the full secrets layer (the Night of Kallos,
> the Scion Shift system, the Arrow's identity, named Quint Bond members). This is why the
> repository is **private**. Do not surface this content in player-facing output, and do not
> make the repo public while these files are present.

## Read order
1. **`sideros_handoff_master.md`** — **the authoritative v18 continuity file.** Single best
   source of truth. Designed to restore full Sideros context in one read. Start here.
2. **`sideros_handoff_supplement.md`** — deep per-civilization, per-Coil-leader, and regional
   campaign detail. Pull when you need to run or write a specific region.
3. **`sideros_aetheric_artifacts_catalogue.md`** — artifact / magic-item catalogue.
4. **`archive/`** — superseded handoffs (v17, v2). Kept for lineage only; where they disagree
   with the master, **the master wins** (it explicitly reconciles them in its final section).

## Authority rule
Where any document in this repo conflicts with `sideros_handoff_master.md` (v18), the master is
current canon — *except* for open items the master itself flags as unresolved (e.g. the
`Concordance`→`Game Master` migration, the Ashélands name, the exact 18th Path). Those need
Matt's confirmation; don't resolve them unilaterally.
