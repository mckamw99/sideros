# Sideros — Player's Comprehensive Guide

An original tabletop roleplaying game set in the world of **Astrolan** (common name: *Sideros*).
5e-SRD-compatible system with a zodiac cosmology.

Open **`index.html`** for the navigable hub linking every page and both editions of the guide.

## Contents

- **`index.html`** — landing hub, categorized links to all material
- **`guide/`** — the Player's Comprehensive Guide (printable binder, `.docx`) + v16 binder edition
- **`pages/`** — 31 HTML reference pages

### Sections
Core Rules · Kindreds · Paths & Weaving · Callings & Orders · Cultures & the Zodiac ·
Bonds, Accords & Companions · Play Systems · Concordance (GM Only)

> **Concordance pages are GM-facing.** `concordance_tools`, `npc_essence`, and `bonds_npcs`
> contain material behind the player/Concordance information boundary. This repository is
> **private** for that reason — keep it private if those pages remain included.

## Canonical terms
Magic use is **Weaving** (a practitioner weaves a **Working**). **Mastery Bonus**, **Loom Tests**,
**Aetheric Channels**, **Concordance** (GM). Practitioners are **Comet-Magi** (sing. Comet-Magus).
