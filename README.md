# Sideros — Player's Comprehensive Guide & Project Canon

An original tabletop roleplaying game set in the world of **Astrolan** (common name: *Sideros*).
5e-SRD-compatible, built on a zodiac cosmology.

This repository is both the **published guide** and a **complete project-context store** — enough
for a future collaborator (human or AI) to understand the entirety of Sideros.

## Start here

- **Reading the game:** open **`index.html`** — the navigable hub linking every page and both
  editions of the guide.
- **Working on the project (esp. as an AI):** read **`CLAUDE.md`** first, then
  **`context/sideros_handoff_master.md`** for the full, authoritative canon.

## ⚠️ Privacy

This repo is **private on purpose.** `context/` and three `pages/` files (`concordance_tools`,
`npc_essence`, `bonds_npcs`) contain GM-only secrets behind the player/GM information boundary —
the Scion Shift system, the Night of Kallos, the Quint Bond identities. **Keep the repo private**
while that material is included. If you ever publish (e.g. GitHub Pages), remove those files first.

## Canonical terms (quick reference)

Magic use is **Weaving** (a practitioner weaves a **Working**). **Mastery Bonus**, **Loom Tests**,
**Aetheric Channels**, **Vitality**, **Guard Rating**. Kindreds are **Comet-Magi** (sing.
Comet-Magus). Full lexicon and the deprecated-term list are in `CLAUDE.md` and the handoff master.
